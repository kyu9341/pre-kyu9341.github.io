#####리스트 원소 랜덤추출

```Python

import random

#중복 허용
count = 2
sampleList = ['a', 'b', 'c', 'd', 'e']
print [random.choice(sampleList) for i in range(count)]

#중복 허용 X
print random.sample(sampleList, count)

```
